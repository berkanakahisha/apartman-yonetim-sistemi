# Apartman Yönetim Paneli

Bu proje tamamen tarayıcı içinde çalışan basit bir apartman yönetim uygulamasıdır.

### Özellikler
- Kullanıcı / daire ekleme
- Aylık aidat – ödeme – kalan borç hesaplama
- PDF rapor oluşturma
- Excel çıktı alma
- Verileri localStorage’da saklama

### Çalıştırma
Sadece `index.html` dosyasını açmanız yeterlidir.

### GitHub Pages Yayınlama
1. Repo oluşturun
2. Dosyaları yükleyin
3. Settings → Pages → "main branch / root" seçin
4. GitHub sizin için canlı linki oluşturacak
